eurostat-map.js — Custom Data Tests

Each page below demonstrates a different map type using entirely made-up data injected via the custom data APIs, without any network calls to Eurostat.

Single-variable maps

  • Choropleth Happiness index (0–100) — statData().setData()
  • Categorical Climate zones — statData().setData() with string values
  • Proportional Symbols Urban population — statData('size').setData()
  • Value by Alpha Renewables share (colour) + green investment (opacity)

Multi-variable maps

  • Bivariate Choropleth GDP per capita × life expectancy
  • Trivariate (Ternary) Choropleth Age structure — youth / working-age / elderly
  • Mushroom Map Exports (left) vs imports (right)

Composition maps

  • Pie Chart Map Electricity generation mix — statPie({ customData })
  • Bar Chart Map Employment by sector — statBar({ customData })
  • Waffle Map Land use — statWaffle({ customData })
  • Stripe Composition Map Transport modal split — statComp({ customData })
  • Coxcomb Map Monthly tourist overnights — statCoxcomb({ customData })

Time-series & flow maps

  • Sparkline Map Annual temperature trend 2018–2024 — statSpark({ customData })
  • Flow Map Trade flows from Germany — flowGraph({ nodes, links })
