An example of a scrollytelling-map using eurostat-map

This example demonstrates how to integrate eurostat-map with scroll-based transitions to create engaging, narrative-driven data visualizations.

Scroll to explore

01 — Initial map

Choropleth view

Here we render a standard map generated using eurostat-map. It shows NUTS-2 regions colored by their GDP per capita index relative to the EU-27 average (= 100).

02 — Highlighting regions

Highlighting regions on the map

As the user scrolls, we highlight specific regions (e.g. high-income regions above 130) in gold while dimming the background regions to guide focus.

03 — Transition to chart

Transitioning the map into a chart

Finally, we morph the complex geographic boundaries of each NUTS-2 region into a standard chart. Each polygon smoothly transitions into a dot on a horizontal dot plot grouped by country, illustrating within-country spread.
